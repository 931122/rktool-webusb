        使用说明与注意事项
        
仅支持 Chrome、Edge 等 Chromium 浏览器，且浏览器必须支持 WebUSB。

        
支持 raw 镜像、压缩镜像和 RKFW。大文件通过流式加载和虚拟文件系统处理，不会一次性占满内存。

    2.

    3. 
          
  3. 设备处于 MASKROM 且当前不是 RKFW 时，需要先选择 Loader。
    4. 
          
  4. 点击“开始刷写”，浏览器弹出授权时允许访问设备。
    5. 
          
  5. 完成后可在当前页直接执行重启、关机或切换存储。
